How does this website work?

  • This website is similar to other wikis, but is focused on games
  • Anyone can write articles about their favorite game for anyone to see
  • The developers of this website will read their articles
  • If they like the article, it will be categorized as an "offical article" and will be changed formatted to appear better and revised
  • This system will motivate users to write high quality articles and will allow readers to know what articles are guaranteed to be high quality
